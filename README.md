# NLP — Весенний семестр 2026

Курс по обработке естественного языка (Natural Language Processing).

Формат:
- 8 семинаров
- летучки на семинарах
- домашнее задание
- командный проект
- промежуточная аттестация

## Оценивание

Итоговая оценка (0–100):

* летучки (4 шт.) — 20%
* домашнее задание — 15%
* командный проект — 25%
* аттестация — 40%

## Используемые инструменты

* Python, Jupyter
* scikit-learn
* spaCy / Natasha
* HuggingFace Transformers
