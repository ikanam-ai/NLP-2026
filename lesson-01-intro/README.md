# Занятие 1 — Введение в NLP

На первом занятии разбираем, что такое NLP, какие задачи он решает и как выглядит базовый пайплайн обработки текста.

---

## Лекция

Темы:

- Задачи NLP: классификация, тональность, NER, поиск
- Базовый NLP-пайплайн: данные → признаки → модель → метрики
- Исторические этапы NLP: TF-IDF → Word2Vec → Transformer → BERT → LLM
- Почему TF-IDF остаётся обязательным baseline
