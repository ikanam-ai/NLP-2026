# Lesson 02 — Word2Vec

## О чём занятие

Переходим от TF-IDF к плотным векторным представлениям слов.

Разбираем:

* почему TF-IDF не учитывает семантику
* как Word2Vec учится по контексту
* Skip-gram и CBOW
* косинусную близость
* как получать эмбеддинг текста через усреднение

## Практика

* обучение Word2Vec на небольшом корпусе
* поиск похожих слов
* простая классификация на усреднённых эмбеддингах
* сравнение с TF-IDF
